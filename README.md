# Core Banking System using Java and SQLite

## About this project
This is a simple banking system made using Java and SQLite. It works like a basic bank app where users can create an account, log in, deposit money, withdraw money, and see their transaction history.

## Features
- Create new bank account  
- Login using account ID and password  
- Deposit money  
- Withdraw money  
- View transaction history  
- Data is saved using SQLite database  

## Technologies used
- Java  
- SQLite  
- JDBC  
- OOP concepts  

## How to run
1. Download or clone this project  
2. Open it in any Java IDE (like IntelliJ or Eclipse)  
3. Add SQLite JDBC driver  
4. Run the `BankingSystem` file  

## What I learned
- Object-Oriented Programming in Java  
- How database connection works using JDBC  
- How real banking systems manage transactions
